# Smart Notes Generator

An AI-powered notes assistant that summarizes long text and generates structured notes using Gemini API.

## Features
- Short summaries
- Key points extraction
- Action items
- Simplified explanations

## Technologies Used
- Python
- Google Gemini API
- Google Colab

## Example Use Cases
- Study notes
- Meeting summaries
- Article summarization
- Quick revision

## Future Improvements
- PDF upload
- Multi-language support
- Voice notes
- Export to PDF
